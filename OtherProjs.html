Game Controller

Ongoing

This goal of this project is to create my own personalized game controller to use with my PC. I am currently working on the 3D model of the controller on Solidworks. The next step is to print a prototype in order to test comfort and size, which I will do using the Dimension 3D Printer available in the UCF TI Innovation Lab. The controller will be coded using an Arduino Uno.

This project was to design and build a high-performance system from the ground up. It served as an excellent experience in research and self training. I learned to identify proper components and gained an understanding of the complexities of a full system integration.
